# 八极拳非遗文化网站
## 项目简介
本项目是基于 HTML + CSS + Bootstrap 5 开发的非遗武术展示网站，整体采用 **永劫无间暗黑武侠风格**，色调沉稳大气，排版统一规整。

网站以八极拳非遗文化为主题，完整包含首页、关于我们、武学作品、非遗传承人四大页面，实现导航与页脚公共组件复用，适配电脑与手机端响应式布局。

## 技术栈
- HTML5 结构搭建
- CSS3 样式美化
- Bootstrap 5 框架
- 组件化引入导航栏、页脚
- 响应式适配移动端

- 全站统一永劫无间暗黑武侠风格
